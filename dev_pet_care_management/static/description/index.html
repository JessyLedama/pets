Pet Care Animal/Bird Management

If you are running a Pet Store which provides services like Pet Boarding, Pet Transportation etc, then using this application you can manage your Pet Store very will in odoo. Create Pet Boarding if you are providing accommodation for pet. Create Pet Service if you are giving services to pet like Bathing, Hair Cut etc. If you are about to transport Pet from one location to another location in the city then create Pet Transportation for it. Appointment is also there So you can create Appointments for Pet Boarding, Service and Transportation to organize your schedule very well. Separate menu for Pet and you can enter all the details related to pet there, So when next time same pet is coming your store then you just have to edit the information if needed. You can generate Pet Agreement for Pet Boarding service. Generate Invoice for Pet Boarding, Pet Service and Pet Transportation. If you are also selling some products from the pet store then you can also create sale order using Pet Retail menu. If you are about to organize any event in pet store then you can send Mass Email to the all the pet Guardians.Various kinds of reports provided for better understanding.

Key Features
  • Separate menu for Pet so you can enter detailed information about each pet
  • Pet Boarding :
    • Create boarding and specify all the services which you have provided to a pet during boarding period
    • Generate Invoice for Pet Boarding
    • Generate Agreement for Pet Boarding
    • Send Boarding details via email to Guardian
  • Pet Service :
    • Create it when you provide services like bathing, hair cut, training etc to a pet
    • Generate Invoice for Pet Service
    • Send Service details via email to Guardian
  • Pet Transportation :
    • Create it when you transfer pet from one location to another location
    • Generate Invoice for Pet Transportation
    • Send Transportation details via email to Guardian
  • Appointment :
    • Appointments are there, so you can manage all the pet services perfectly
    • Create Appointment for Pet Boarding, Pet Service or Pet Transportation
    • Generate Pet Boarding from appointment
    • Generate Pet Service from appointment
    • Generate Pet Transportation from appointment
    • Send appointment details to guardian via email
  • Agreement :
    • Separate menu to manage Pet Boarding Agreements
    • Separate menu for Agreement Documents also, so you can find all the documents of agreement in one place
    • Send Agreement via email to Guardian
  • Contacts : Mange all the Guardians of pet under this menu
  • Retail :
    • If you also selling some products from pet store then you can use Retail menu to generate sale order
    • Find Products menu under Retail, here you can see only products which are related to Pet
  • Mass Mailing :
    • Send email to all the guardians on a single click
    • Suppose you are going to organize free medical checkup camp for pets then you do not have to send invitation to each guardian separately. Just use mass mailing to send email to all the guardian on a single click
